# 🚀 Smart Hovercraft for Flood Rescue (ESP32 + Bluetooth)

> A Bluetooth-controlled hovercraft designed to deliver **food and medical supplies during floods**.

---

## 📌 Overview

This project uses an **ESP32 microcontroller** to control a hovercraft with:

- Lift motor (for hovering)
- Thrust motor (for movement)
- Wireless Bluetooth control

---

## ⚙️ Features

- 🔵 Full system ON/OFF control  
- 🚁 Automatic lift activation  
- 🟢 Adjustable thrust speed  
- 📡 Bluetooth-based control  
- ⚠️ Emergency STOP  

---

## 🎮 Control Commands

| Command | Function |
|--------|---------|
| `A` | System ON |
| `P` | System OFF |
| `L` | Thrust ON |
| `R` | Thrust OFF |
| `T` | Low Speed |
| `C` | Medium Speed |
| `X` | High Speed |
| `S` | Emergency STOP |

---

## 📦 Components Used

- ESP32
- BLDC Motors
- ESC Controllers
- LiPo Battery

---

## 🧠 Applications

- Flood rescue  
- Medical delivery  
- Remote transport
